# AGENTS.md

## 项目目标
把 Cell 做成桌面应用原型，而不是网页原型。

## 当前阶段目标
先完成一个可运行的 Electron 桌面应用壳，并把现有的 React 原型迁移为桌面应用结构。

## 形态要求
这是桌面灵感捕捉工具，不是网页，不是官网，也不是看板后台。

## 第一优先级
1. 单窗口桌面应用结构
2. 强化悬浮入口 / 快速捕捉
3. 母体空间作为主工作区
4. 详情与整理作为浮层，而不是网页页面切换

## 技术规则
- 保留 React + TypeScript
- 使用 Electron Forge 当前模板
- 不接后端
- 不接数据库
- 不做登录
- 只做桌面原型
- 保持代码清晰，不要过度工程化

## 设计规则
- 中文界面
- 深色
- 生物感 + 系统感
- 不要像 SaaS 后台
- 不要像网页首页
- 不要像白板工具
